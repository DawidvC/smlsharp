This is a library for easy scripting.

The objective of this library is to attract other scripting languages users, who are not familiar with functional language but who have interest in it.

This library shows that they can write ML codes of the same functionality as those written in their familiar languages with no difficulties. Interface of this library is in imperative style rather than functional style, in order to make levels of entry lower for newcomers.

This library provides the following features.

  • text manipulation
    • word, field, and line manipulation
    • string matching and substitution
    • regular expression matching and substitution
  • easy access to standard I/O
    • 'print' and 'inputLine' on standard I/O are available at top level by default.
  • cooperation with shell
    • easy access to environment variable
    • easy access to command line argument command invocation
Many of them are just thin wrappers of functions defined in Basis library.

The motto of this library is 'easy coding' rather than 'clean coding'. The interface needs not be neat. It should gather various features greedily.

This module aims to provide the same easiness as the functions which are available at the top-level in other scripting languages.
